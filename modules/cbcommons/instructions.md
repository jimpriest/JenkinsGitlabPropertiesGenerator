#INSTRUCTIONS

Just drop into your **modules** folder or use the box-cli to install

`box install cbcommons`

All model objects are autowired with a `@cbcommons` namespace.

- `DateUtils@cbcommons`
- `FileUtils@cbcommons`
- `JVMUtils@cbcommons`
- `QueryHelper@cbcommons`
- `Zip@cbcommons`
- `Utilities@cbcommons`
